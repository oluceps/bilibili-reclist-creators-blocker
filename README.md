# 批量拉黑「接下来播放」列表内容作者 - Bilibili

发现对同类视频下面的推荐列表内容创作者进行屏蔽似乎可以高效过滤垃圾内容

使用方式：点击 Tampermonkey 浏览器插件并 Create a new script，黏贴 `main.js` 中的内容进去并保存，并刷新b站界面，最后点击按钮。
